Features
--------

- Automated SQLite table creation from data
- Support various data types of record(s) insertion into a table:
    - ``dict``
    - ``namedtuple``
    - ``list``
    - ``tuple``
- Create table(s) from:
    - CSV file/text
    - JSON file/text
    - `TableData` instance loaded by `pytablereader <https://github.com/thombashi/pytablereader>`__
