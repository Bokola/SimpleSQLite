Feature
-------

- Automatic table creation from data
- Support various data types of record(s) insertion into a table:
    - dictionary
    - namedtuple
    - list
    - tuple
- Create table(s) from:
    - CSV file/text
    - JSON file/text
    - Microsoft Excel |TM| file
    - `Google Sheets <https://www.google.com/intl/en_us/sheets/about/>`_
